Đảm bảo bạn có python phiên bản mới nhất mà npm mới nhất!

Chạy backend:
  - cd backend
  - venv/scripts/activate
  - python manage.py runserver


Chạy frontend:
  - cd frontend
  - npm install
  - npm run dev
